# retrogames
Collection of self build retro games.


Emulators i used: <br>
- 6502 asm: https://skilldrick.github.io/easy6502/ <br>
- GB: Bgb64 https://bgb.bircd.org/ <br>
- NDS: deSmuMe http://desmume.org/ <br>
